# Welcome

Welcome to the documentation for the [rumblpak Helm charts](https://github.com/rumblpak/helm-charts) project.

## Getting started

[Helm](https://helm.sh) must be installed to use the charts in this repository.

Refer to Helm's [documentation](https://helm.sh/docs/) to get started.

## Installation

```sh
helm repo add rumblpak http://rumblpak.github.io/helm-charts/
```

You can then run `helm search repo rumblpak` to search the charts.
